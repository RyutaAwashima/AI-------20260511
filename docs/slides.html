Mythos — Concrete Steps

実際にやったことの例

1

「ストーリー生成アプリを作って」と Copilot に話しかける

HTML・CSS・JavaScript のひな形を一瞬で生成。ゼロからコードを書かなくていい。

2

GAS で Gemini API プロキシを作成・デプロイ

サーバーコード（gas_proxy.gs）も Copilot が生成。デプロイすれば無料サーバーが立ち上がる。

3

「スプシにデータを保存したい」→ スキーマ設計を依頼

Google Sheets を DB として使う設計をCopilotと一緒に決める。11列のスキーマを自動構築。

4

バグが出たら「このエラーを直して」と貼るだけ

エラーメッセージを Copilot に見せると原因と修正コードを提示してくれる。

5

GitHub にプッシュ → 誰でもクローンして使える

コードを共有するだけで他の端末や仲間が同じツールをすぐ使える。
